The `@earnit/sdk` package exposes a single `EarnIT` client.

```ts
import { EarnIT } from "@earnit/sdk";

const earnit = new EarnIT({
  apiKey: process.env.EARNIT_API_KEY!,
  // optional
  baseUrl: "https://api.earnit.xyz/v1",
});
```

## Constructor options

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `apiKey` | `string` | — | Your `ek_pub_` or `ek_` key. Required. |
| `baseUrl` | `string` | `https://api.earnit.xyz/v1` | API base URL. |

## Conventions

- All **amounts are decimal strings** in the asset's units (e.g. `"150.00"`), never floats.
- All methods return promises and throw a typed `EarnitError` on failure — see [Errors](/docs/api-reference/errors).
- `asset` is one of `"USDC" | "cNGN" | "EURC"`.
